# Roadmap

This document lists the planned features and improvements for the project.

## Planned Features
- [x] Improve error messages with line numbers and better context.
- [x] Assignment operator (`=`).
- [x] Add `List` data type.
- [x] Add loops (for, while).
- [x] Add negative numbers.
- [x] Add modulo operator.
- [x] Add comments.
- [x] Add `Map` (Dictionary) data type.
- [x] Add `String` indexing.
- [x] Change rule name from string into enum.
- [ ] Refactor:
    - [x] Move out built in functions into separate files
    - [ ] Rearrange file structure
    - [ ] Fix warrnings
    - [ ] Remove old comments
- [ ] Add support for structs/classes and object-oriented programming.
- [ ] Add more built-in functions (e.g., `len`, `input`, string manipulation).
- [ ] Generate VS Code / Vim syntax highlighting extensions from the grammar.

## Completed Features
- [x] Basic arithmetic operations (`+`, `-`, `*`, `/`).
- [x] Comparison operators (`==`, `!=`, `<`, `>`).
- [x] Logical operators (`&&`, `||`, `!`).
- [x] Functions (definition and calling).
- [x] Control flow (`if`, `else`).
- [x] Basic types (`Int`, `Float`, `String`, `Bool`).
